### public static void main(String[] args)
This is called the main method.
It is where Java starts executing your program
Let's understand each keyword.

**public:**
It means the JVM can access this method from anywhere.
Without public, the JVM may not be able to start the program.
**static:**
Normally, to use a method, we create an object.
Example:
Student s = new Student();
But **the main() method is static, so the JVM can call it without creating an object.**
**void** means the method does not return any value.
Example:
public static void main(...)
After finishing its work, it doesn't send anything back.
main
This is the entry point of every Java program.

When you run a Java program, the JVM first looks for:
**main()**
If it cannot find it, the program won't run.
**String[] args**

This is an array of strings used to receive command-line arguments.

Simple example:

If you run:

java HelloWorld Naseema
Then:
args[0]
contains:
Naseema
For beginners, you usually won't use args often.

**System**
System is a built-in Java class that provides useful features like input and output.

**out** is the standard output stream.

"Display the output on the computer screen."
**println means:**

**; (Semicolon)**
System.out.println("Hello");
The semicolon (;) marks the end of a statement in Java.
If you forget it, the program won't compile.
